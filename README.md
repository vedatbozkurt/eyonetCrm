## About

Laravel based multitenant (Admin and Superadmin login) web application for customer relationship management. 


####Installation

`$ composer install or php composer.phar install`

`$ php artisan key:generate`

`$ php artisan migrate`

`$ php artisan db:seed`

`$ php artisan serve`
